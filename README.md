# Admin Healthcare Frontend

A modern, responsive admin portal frontend for healthcare management built with React, Vite, and Tailwind CSS.

## Features

- **Dashboard** with real-time statistics
- **User Management** - View and manage doctors and patients
- **Appointments** - Full appointment management with pagination
- **Billing** - Invoice management with monthly filtering
- **Prescriptions** - Medication tracking and management
- **Test Results** - Medical records viewing
- **Messages** - Patient-doctor communication threads
- **Audit Logs** - System activity tracking

## Tech Stack

- React 18 + Vite 7.2.2
- Tailwind CSS
- React Router
- Axios
- Recharts

## Installation

\`\`\`bash
git clone https://github.com/sairamreddy7/Admin-Healthapp-Frontend.git
cd Admin-Healthapp-Frontend
npm install
npm run dev
\`\`\`

## Environment Variables

Create a \`.env\` file:
\`\`\`properties
VITE_API_URL=https://your-api-url.com/api
\`\`\`
